# Minimal Neovim Configuration

## Introduction

**File Structure**
```
nvim
├── init.lua
├── lua
│   ├── keymaps.lua
│   ├── options.lua
│   └── plugins
│       ├── keymaps.lua
│       └── lazy.lua
└── README.md
```

## Install

**SSH**
```bash
git clone git@github.com:OvidioCalvet/my-neovim-config.git
```

**HTTP**
```bash
git clone https://github.com/OvidioCalvet/my-neovim-config.git
```

## Uninstall

```bash
rm -rf ~/.config/nvim
mkdir ~/.config/nvim
```
